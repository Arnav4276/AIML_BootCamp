# AIML Bootcamp 🚀

Welcome to the AIML Bootcamp repository! This comprehensive program takes you from programming basics to advanced Large Language Models (LLMs) through hands-on learning.

**What's Covered:**

* **Python & Data Science:** Master Python fundamentals, NumPy, Pandas, data cleaning, and visualization.
* **Machine Learning Core:** Understand models, features, predictions, and the essential math behind ML (vectors, distance, learning rates).
* **Deep Learning & LLMs:** Explore sequence models (LSTMs) and dive deep into Transformers, attention mechanisms, and how modern LLMs operate.
* **Mini ML Project:** Apply your skills by building an AI-assisted portfolio generator using LLM APIs.

Join us on this exciting journey into AI!
